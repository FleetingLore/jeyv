# 二次函数的图像与性质

最简单的二次函数应当是 $f(x) = x^2$ , 对应的图形的解析式为 $y = x^2$ ,其中 $y$ 轴为纵轴.

通过平移,可以得到处于不同位置的二次函数图像.
通过缩放,可以得到不同形状的二次函数图像.

此处不介绍旋转变换的事.

## 一、二次函数的解析式

按一般形式,二次函数形如

$$f(x) = ax^2 + bx + c\ (a \ne 0) \tag{1.3.2-1}$$

另有顶点式

$$f(x) = a(x-h)^2 + k\ (a \ne 0) \tag{1.3.2-2}$$

有 $h$ 、 $k$ 满足 $h = -\frac{b}{2a}$ , $k = c - \frac{b^2}{4a}$ .
这种形式直观地体现了其对应函数的最值和对称性,这在先前的《方程与方程组》篇幅有所表述.

另有

$$f(x) = a(x - x_1)(x - x_2)\ (a \ne 0) \tag{1.3.3-3}$$
