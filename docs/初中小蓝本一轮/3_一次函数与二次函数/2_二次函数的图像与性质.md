# 二次函数的图像与性质

## 一、二次函数的解析式

按一般形式,二次函数形如

$$f(x) = ax^2 + bx + c\ (a \ne 0) \tag{1.3.2-1(a)}$$

另有顶点式

$$f(x) = a(x-h)^2 + k\ (a \ne 0) \tag{1.3.2-1(b)}$$

有 $h$ 、 $k$ 满足 $h = -\frac{b}{2a}$ , $k = \frac{4ac-b^2}{4a}$ .

另有

$$f(x) = a(x - x_1)(x - x_2)\ (a \ne 0) \tag{1.3.3-1(c)}$$

其中对称轴方程为 $x = \frac{x_1 + x_2}{2}$ ,顶点坐标为 $\left(\frac{x_1 + x_2}{2}, \frac{-a(x_1 - x_2)^2}{4}\right)$.

这种公式,自己脑补一遍推导过程,再构思一些相关的例子,大概就能记住了.

书上说的那个,用三点确定二次函数解析式的公式,不作证明.
如果列方程硬算,我试过,写了两页纸.
比较快的方法是,参考线性系统的叠加性,注意到函数空间到函数值的映射式线性的.
这个叠加性,在之后大家会学到的微分方程,也能用到.
另,按照 deepseek 的说法,这是拉格朗日插值公式在二次函数形式下的体现.

一次函数的图形是过 $(0,\ b)$ , $\left(-\frac{b}{k},\ 0\right)$ 两点的直线.
