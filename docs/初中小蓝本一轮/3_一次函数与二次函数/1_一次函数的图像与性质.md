# 一次函数的图像与性质

---
> ***一次函数***
>
> $$y = k x + b\ (k \neq 0)$$

一次函数的图形是过 $(0,\ b)$ , $\left(-\frac{b}{k},\ 0\right)$ 两点的直线.

正比例函数的图形关于原点中心对称.
