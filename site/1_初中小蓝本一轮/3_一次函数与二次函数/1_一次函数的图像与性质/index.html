    一次函数的图像与性质

    一次函数

    \[y = k x + b\ (k \neq 0)\]

    一次函数的图形是过 \((0,\ b)\) , \(\left(-\frac{b}{k},\ 0\right)\) 两点的直线.

    正比例函数的图形关于原点中心对称.

    \(y = k x + b\)

    $\boxed 1\ \ $ \(k = 0\) | 常函数

    $\boxed 1\ \ $ \(b = 0\)

    经过原点的水平直线.

    $\boxed 2\ \ $ \(b > 0\)

    在 \(x\) 轴上方的水平直线

    $\boxed 3\ \ $ \(b < 0\)

    在 \(x\) 轴下方的水平直线

    $\boxed 2\ \ $ \(k > 0\)

    $\boxed 1\ \ $ \(b = 0\) | 上升的正比例函数

    经过原点的上升直线.

    $\boxed 2\ \ $ \(b > 0\)

    与 \(x = 0\) 的交点在原点上方的上升直线

    $\boxed 3\ \ $ \(b < 0\)

    与 \(x = 0\) 的交点在原点下方的上升直线

    $\boxed 3\ \ $ \(k < 0\)

    $\boxed 1\ \ $ \(b = 0\) | 下降的正比例函数

    经过原点的下降直线.

    $\boxed 2\ \ $ \(b > 0\)

    与 \(x = 0\) 的交点在原点上方的下降直线

    $\boxed 3\ \ $ \(b < 0\)

    与 \(x = 0\) 的交点在原点上方的下降直线
